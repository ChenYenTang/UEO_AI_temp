# Utils 模組

`utils` 目錄包含了一系列輔助工具模組，這些模組提供了模型訓練流程中所需的核心元件，如損失函數、優化器和學習率排程器。將這些元件模組化有助於保持主要訓練邏輯的整潔與可擴展性。

## 檔案結構與功能

### `__init__.py`

此檔案將 `utils` 目錄標示為一個 Python 套件，並從各個子模組中導出可用的元件列表（如 `LOSS_LIST`）和建構函式（如 `build_loss`），使得上層應用可以方便地引用。

### `losses.py`

-   **功能**: 定義和管理模型訓練中可用的損失函數。
-   **職責**: 提供一個 `build_loss` 函式，可以根據傳入的名稱字串（如 "MSE"）回傳對應的 PyTorch 損失函數實例 (`torch.nn.MSELoss`)。

### `optimizers.py`

-   **功能**: 定義和管理模型訓練中可用的優化器。
-   **職責**: 提供一個 `build_optimizer` 函式，可以根據傳入的名稱字串（如 "Adam"）和模型參數，回傳對應的 PyTorch 優化器實例 (`torch.optim.Adam`)。

### `schedulers.py`

-   **功能**: 定義和管理模型訓練中可用的學習率排程器。
-   **職責**: 提供一個 `build_scheduler` 函式，可以根據傳入的名稱字串（如 "StepLR"）和優化器，回傳對應的 PyTorch 學習率排程器實例。
